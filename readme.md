# SplitMark

A minimalist Quill based text editor with markdown support

## Keyboard Shortcuts
* Control Shift C: View word count
* Control +/-: Edit font size
* Control Z: Undo
* Control Y: Redo

## Supported Markdown Syntax

1. 1 DEPTH
2. 2 DEPTH
 1. 2 SUB-DEPTH
 2. 2 SUB-DEPTH

string to **strong** and string to `hightlight`

```
# HEADER

This is ~~code-block~~

TODO

[x] Sleep
[ ] More sleep


```
